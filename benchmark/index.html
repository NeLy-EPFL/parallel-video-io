    Benchmarks

    PVIO is benchmarked against PyAV, OpenCV, Decord, TorchCodec, and NVIDIA DALI across three tasks. Results below are generated by benchmark/run_all.py and embedded here automatically at docs-build time. To regenerate locally:

    uv run python -m benchmark.run_all
mkdocs gh-deploy

    See benchmark/README.md for full methodology, metric definitions, and backend notes.
